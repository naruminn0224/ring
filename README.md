# IRIAM Icon Ring Creator - デプロイガイド

このツールはビルド不要な静的サイト（HTML/CSS/JS）として構成されているため、非常に簡単に無料でインターネット上に公開できます。

## おすすめの公開方法

### 1. Vercel / Netlify (最も簡単)
初心者の方に最もおすすめの方法です。
1. [Vercel](https://vercel.com/) または [Netlify](https://www.netlify.com/) にアクセス。
2. 作成した 3 つのファイル（`index.html`, `style.css`, `main.js`）を 1 つのフォルダにまとめます。
3. そのフォルダをサイト上のアップロードエリアに「ドラッグ＆ドロップ」するだけで公開完了です。

### 2. GitHub Pages (推奨)
コードを管理しながら公開したい場合に最適です。
1. GitHub で新しいリポジトリを作成。
2. ファイルをアップロード（Push）。
3. 設定（Settings > Pages）から公開設定を有効にします。

### 3. Cloudflare Pages
高速で安定した公開が可能です。
1. GitHub と連携するか、直接ファイルをアップロードして公開できます。

---

## 注意点
- **著作権**: 素材画像や透過PNGの取り扱いについては、利用規約などを明記することをお勧めします。
- **CDN**: このツールは `fabric.js` を CDN 経由で読み込んでいるため、オフラインでは動作しません。公開した状態（オンライン）で使用してください。
